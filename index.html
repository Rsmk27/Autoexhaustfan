The Challenge We're Solving

Traditional ventilation relies on manual control, leading to human error and safety risks. We're changing that.

Safety Risks

Undetected gas leaks or smoke accumulation can lead to fires and health hazards. Immediate response is critical.

Human Error

Forgetting to turn on fans is common. Manual systems fail when people forget, compromising safety.

Energy Waste

Fans left running unnecessarily waste power. Automated systems run only when needed, saving energy.

Firmware Code

Optimized C++code for the Arduino platform to handle real-time sensor data processing and actuator control.

constintMQ2_Pin=A0;
        constintFan=7;
        constintThreshold=300;

        voidsetup() {
            pinMode(MQ2_Pin, INPUT);
            pinMode(Fan, OUTPUT);
            Serial.begin(9600);
        }

        voidloop() {
            intsmokeLevel=analogRead(MQ2_Pin);
            Serial.print("Smoke Level: " );
            Serial.println(smokeLevel);

            if(smokeLevel > Threshold) {
                digitalWrite(Fan, HIGH);
                 // Activate Fan
            }

            else {
                digitalWrite(Fan, LOW);
                 // Deactivate Fan
            }

            delay(1000);
        }
